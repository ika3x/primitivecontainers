# ArsenalOfCp - "AC"

Java標準コレクションの「型安全」という恩恵を受けつつ、パフォーマンスを最大限に引き出すための高機能・軽量ライブラリです。

競技プログラミングの過酷な制約下での使用を想定して設計されていますが、メモリ効率とキャッシュ局所性を重視した設計のため、大規模データの処理やリアルタイム性が求められるシステム開発の実務においても強力なツールとなります。

## なぜこのライブラリか？

* **Zero Dependency**: Java標準ライブラリのみで構成。余計な依存関係をプロジェクトに追加する必要はありません。
* **Primitive Optimization**: できる限り`ArrayList<Long>` のようなオートボクシングによるメモリ浪費やGCの負荷を回避した仕様です。
* **Performance First**: CPUキャッシュ効率を意識したデータ構造（ブロック分割法など）を採用しており、`TreeSet` 等の標準APIを上回るパフォーマンスを発揮します。
